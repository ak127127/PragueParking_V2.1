# Prague Parking 2.1 (Console, English)

Console app fulfilling the assignment requirements.

## Run
- Open `PragueParking_V2.1.sln` in Visual Studio 2022
- Set `PragueParking.App` as startup project
- Press F5

Or via CLI:
```
dotnet build
dotnet run --project src/PragueParking.App
```

Use menu option **Reload config/prices** to re-read `assets/config.json` and `assets/prices.txt`.
